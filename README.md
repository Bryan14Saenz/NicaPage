# Nicaragua Conócenos

Bienvenido al repositorio de **"Nicaragua Conócenos"**, un sitio web informativo y visualmente atractivo donde podrás explorar lo mejor de Nicaragua: su cultura, historia, gastronomía, turismo y curiosidades.

## 📌 Descripción

Este proyecto tiene como objetivo mostrar al mundo las maravillas de Nicaragua a través de una página estática desarrollada con HTML, CSS y recursos multimedia. Ideal para quienes quieren aprender sobre este hermoso país o practicar el desarrollo web creando una web informativa con un diseño responsivo.

## 🧩 Tecnologías utilizadas

- HTML5
- CSS3
- Diseño responsivo
- Recursos multimedia (imágenes, videos, íconos)

## 🧭 Estructura del sitio

El sitio se divide en las siguientes secciones:

- **Inicio**: Imagen destacada y bienvenida.
- **Turismo**: Lugares turísticos de distintas ciudades nicaragüenses.
- **Gastronomía**: Platos típicos del país.
- **Historia**: Línea del tiempo de eventos clave de Nicaragua.
- **Curiosidades**: Hechos únicos y divertidos del país.
- **Contacto**: Enlaces a redes sociales y correo de contacto.
